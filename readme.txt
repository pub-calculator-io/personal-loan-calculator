After factoring the fees, insurance, and interest of a personal loan, this free personal loan calculator computes the monthly payment, true loan cost, and Annual percentage rate (APR).

[https://www.calculator.io/personal-loan-calculator/](https://www.calculator.io/personal-loan-calculator/)

== Usage ==

Add the Personal Loan Calculator shortcode to your page, post or sidebar:

`[ci_personal_loan_calculator]`

Add the following code to your template where you would like the Personal Loan Calculator to appear:

`<?php display_ci_personal_loan_calculator(); ?>`

== Installation ==

1. Upload the Personal Loan Calculator /ci_personal_loan_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Personal Loan Calculator plugin through the "Plugins" menu in WordPress.
